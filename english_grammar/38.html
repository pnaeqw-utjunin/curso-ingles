Última actualización 30/11/2025

Bienvenidos al English Grammar Podcast. Les saluda Emily y Carlos. Hoy vamos a estudiar un tema fundamental en la gramática del inglés: "If I do... and if I did..." - "Si lo hago... y si lo hiciera...". Este tema es clave para hablar de situaciones reales e irreales en el presente y el futuro.

Empecemos con la estructura y uso de estas frases condicionales. Primero, usamos "if + present simple" - "si + presente simple" para hablar de situaciones reales o posibles.

Por ejemplo:

  • If we take the bus, it will be cheaper - Si tomamos el autobús, será más barato.
  • If I see Anna, I will tell her about the meeting - Si veo a Anna, le diré sobre la reunión.
  • If it rains, we will stay at home - Si llueve, nos quedaremos en casa.


Ahora, si la situación no es real o no esperamos que ocurra, usamos "if + past simple" - "si + pasado simple". Por ejemplo:

  • If we took the bus, it would be cheaper - Si tomáramos el autobús, sería más barato.
  • If I had a car, I would drive to work - Si tuviera un coche, iría al trabajo manejando.
  • If she knew the answer, she would tell us - Si ella supiera la respuesta, nos lo diría.


Observa la diferencia:

  • If you find it, can you call me? - Si lo encuentras, ¿puedes llamarme?
  • If you found a wallet in the street, what would you do with it? - Si encontraras una cartera en la calle, ¿qué harías con ella?


En el segundo caso, estamos hablando de una situación hipotética o imaginaria. También podemos usar "could" y "might" en lugar de "would" para expresar posibilidad o habilidad:

  • If I won a lot of money, I might buy a house - Si ganara mucho dinero, tal vez compraría una casa.
  • If we studied more, we could pass the exam - Si estudiáramos más, podríamos aprobar el examen.


Ahora vamos a poner en práctica todo lo aprendido.

Escucha estas frases y observa el uso de los condicionales.

  • If you found a wallet in the street, what would you do with it? - Si encontraras una cartera en la calle, ¿qué harías con ella?
  • If we took the train, it would be quicker - Si tomáramos el tren, sería más rápido.
  • If I lost my passport, I wouldn’t know what to do - Si perdiera mi pasaporte, no sabría qué hacer.
  • If he knew the truth, he wouldn’t lie - Si él supiera la verdad, no mentiría.
  • If they offered me the job, I would accept - Si me ofrecieran el trabajo, lo aceptaría.


Completemos algunas frases:

  • If you dropped that vase, it would break into small pieces - Si dejaras caer ese florero, se rompería en pedazos pequeños.
  • If I saw Sarah, I’d tell her to call you - Si viera a Sarah, le diría que te llame.
  • If you went right at the end of this street, you’d see a bank on your left - Si giraras a la derecha al final de esta calle, verías un banco a tu izquierda.
  • If I had wings, I could fly - Si tuviera alas, podría volar.
  • If we lived near the beach, we might go swimming every day - Si viviéramos cerca de la playa, tal vez nadaríamos todos los días.


Ahora unas preguntas hipotéticas para reflexionar:

  • What would you do if you won a lot of money? - ¿Qué harías si ganaras mucho dinero?
  • What would happen if somebody pressed that button? - ¿Qué pasaría si alguien presionara ese botón?
  • What would you say if I told you a secret? - ¿Qué dirías si te contara un secreto?
  • What would you do if you were invisible for one day? - ¿Qué harías si fueras invisible por un día?


Completemos frases condicionales:

  • If I had a party, who would you invite? - Si hiciera una fiesta, ¿a quién invitarías?
  • If I asked you for help, would you do it? - Si te pidiera ayuda, ¿lo harías?
  • If somebody gave me $20,000, I’d have a long holiday - Si alguien me diera $20,000, tendría unas largas vacaciones.
  • If I didn’t want to go yet, I’d not come - Si aún no quisiera ir, no vendría.
  • If she met a famous actor, she’d take a photo - Si conociera a un actor famoso, se tomaría una foto.


Finalmente, algunas frases para practicar el inicio con "If...":

  • If she took the 10:30 train, she’d arrive too early - Si ella tomara el tren de las 10:30, llegaría demasiado temprano.
  • If Kevin applied for the job, he’d get it - Si Kevin solicitara el trabajo, lo conseguiría.
  • If we had more time, we could visit the museum - Si tuviéramos más tiempo, podríamos visitar el museo.
  • If I were you, I’d talk to the manager - Si yo fuera tú, hablaría con el gerente.
  • If it stopped raining, we could go out - Si dejara de llover, podríamos salir.
